`SetupLogging(dataDir string, level string) *slog.Logger`:
- Build log path: filepath.Join(dataDir, "ems_watcher.log")
- Create lumberjack.Logger{Filename: logPath, MaxSize: 2, MaxBackups: 5, Compress: false}
- Create a custom slog.Handler that formats as "2006-01-02 15:04:05 [LEVEL] message" in UTC
  - Use slog.NewTextHandler as baseline but override with a custom Handler wrapping io.MultiWriter(jack, os.Stdout)
  - The simplest approach: implement a thin `emsHandler` struct satisfying `slog.Handler` that writes the fixed format to the io.MultiWriter
  - Alternatively: use `slog.NewTextHandler` with `slog.HandlerOptions{ReplaceAttr: ...}` but the format won't match exactly — use custom handler for exact match
- Parse level string: map "DEBUG"->slog.LevelDebug, "INFO"->slog.LevelInfo, "WARNING"/"WARN"->slog.LevelWarn, "ERROR"->slog.LevelError; default INFO
- Return `slog.New(handler)`

Custom format (exact string to produce):
`2026-04-20 14:32:01 [INFO] message text here`
Timestamp: `time.Now().UTC().Format("2006-01-02 15:04:05")`
Level: `[INFO]`, `[DEBUG]`, `[WARNING]`, `[ERROR]`
